# AI-Generated Essays Detection

This project aims to detect whether an essay is written by a human or generated
by artificial intelligence using deep learning and NLP techniques.

## Models Used
- LSTM
- DistilBERT
- ELECTRA Small

## Dataset
- Human vs AI-Generated Essays dataset
- Balanced dataset (Human-written vs AI-generated)

## Preprocessing
- Lowercasing
- Punctuation removal
- Stopword removal
- Lemmatization

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Tools & Libraries
- Python
- TensorFlow / Keras
- HuggingFace Transformers
- scikit-learn
- NLTK
- spaCy
- LIME
- SHAP
